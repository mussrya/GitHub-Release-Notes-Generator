About

GitHub Release Notes Generator is exactly what the name suggests, a release notes generator for GitHub. This tool was originally created to reduce the effort in producing release notes for a particular milestone.


After logging in with your GitHub account and authorizing the application access to information, several calls are made to GitHub using their current API to display your repositories, milestones and then ultimately the required issue information to produce release notes.


Providing your information is correct and structed in a way which you require then this tool could save time each release. A print button is provided which can be clicked and saved as a PDF which could then be emailed out or hosted somewhere for internal and or external users.
